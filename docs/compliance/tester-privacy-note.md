# Productivitree Beta Tester Privacy Note

Effective date: 2026-03-02

## What Productivitree stores
- Core tree data (roots, trunk segments, projects, stages).
- Theme preferences.
- Tree health state.
- Onboarding completion state.
- Optional passion-test result.

Storage location: browser localStorage (local device).

## AI usage disclosure
- Passion-test answers are sent only if the user explicitly enables AI reflection consent.
- AI processing is performed by a server-side Netlify Function.
- The Gemini API key is not exposed in client-side code.

## Data retention
- Local browser data remains until user clears it or uses in-app delete/reset controls.
- No analytics collection is enabled by default in MVP testing mode.

## User controls
- Copy/export full data payload.
- Reset demo data.
- Delete all stored data.

## Beta contact
- Test feedback is collected through the repository issue-based feedback link.
